# jenkins-job-timings-visualisations
## Description
Scripts to analyse a sequence of Jenkins jobs and create visualisations (e.g. Gantt chart)

## Requirements 
Python 2.7+, Plotly (pip install plotly)

## Usage
python jenkins-job-gantt.py
